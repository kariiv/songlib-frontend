# Songs with Tabs collection app

New version of forntend with updated backend.

## Features

- SONG: Title, Author, Lyrics, Link, Lang, Tags, Rank(difficulty), Time
- Create/Edit/Delete Song
- Create/Edit/Delete Tag
- Local history
- Autoscroll
- Transpose
- Search
- Video collapse card
- Playlist strategies:
  - All
  - Artist
  - Tag
  - Link
  - Rank
  - History
- Sorting
  - Artist
  - Title
  - Rank
  - Time
  - Difficulty
